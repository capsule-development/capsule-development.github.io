Capsule

An all new package manager for iOS

About

Capsule is an upcoming package manager for your jailbroken devices. Developed with care by the @capsuleios team!

We aim to bring a modern, fast, and simple package manager that works alongside any jailbreak and other package managers.

Features

Created in SwiftUI

With iOS 13 design kept in mind, we strive to provide a sleek and intuitive interface.


You can also setup a Capsule Account to sync repositories and tweaks across devices, control your package manager remotely, and more.
